The main screen

On the main screen, images can be uploaded. These will then be converted to a suitable format to allow deep zooming. This conversion progress may take some time.

When an image has been processed, it will appear in the Uploaded images list. From there, it can be opened by clicking its thumbnail. This will take you to the annotation screen.

The annotation screen

The annotation screen shows the selected image, initially in full screen. In the bottom-left corner, the name of the selected image is shown.

The viewer can be moved around by clicking and dragging the mouse cursor, or by using the arrow keys.

You can zoom in by scrolling, or using the zoom controls in the bottom-right corner of the viewer.

In the top-left corner is a list of image controls:

Hide/show all buttons
Show the image in a split viewer, for easier viewing when annotating.
Open the mask manager, where segmentation masks can be added and managed.
The mask manager

Clicking the Set mask button opens the mask manager. From there, a segmentation mask can be uploaded. This should not take long.

Once a mask has been uploaded, it can be applied by clicking its thumbnail. This shows the mask in an overlay on the image.

It also enables several editing tools in the top-left corner:

Draw a new segment using a polygon drawing tool.
Cut (parts of) segments using a scissor tool.
Simplify all segments' polygons. This will reduce the number of vertices and edges.
Remove all inner rings (holes) from segments.
